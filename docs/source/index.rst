f3kdb
=====

**f3kdb** (a.k.a. flash3kyuu_deband) is a deband library and `Avisynth <http://sourceforge.net/p/avisynth2/>`_ / `Vapoursynth <http://www.vapoursynth.com/>`_ filter, ported from `an aviutl plugin <http://www.geocities.jp/flash3kyuu/auf/banding17.zip>`_.

It works by replacing banded pixels with average value of referenced pixels, and optionally add grain (random dithering) to them.
